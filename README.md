# react-wize-brand
React components for Wizeline logos

# Install

`yarn add https://github.com/wizeline/react-wize-brand#main`

# Props:

| Name      | Type    | Default     | Description             |
| --------- | ------- | ----------- | ----------------------- |
| **svg**   | boolean | **false**   | Show logo as SVG or PNG |
| **width** | string  | **'100px'** | Set logo's width        |
